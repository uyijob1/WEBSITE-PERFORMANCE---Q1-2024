🚀 Website Performance Dashboard – Q1 2024

This project presents an interactive dashboard built in Power BI Desktop to analyse website performance across the first quarter of 2024.

The dashboard provides insights into user behaviour, traffic sources, and performance trends using data modelling, DAX calculations, and effective data visualisation techniques.

📊 Key Features

KPI Overview
Total Users: 5,059
Total New Users: 4,467
Total Sessions: 8,188

Channel Performance Analysis
Identifies top-performing channels (Direct, Social, Organic Search)
Highlights underperforming channels (Email, Paid Search)

Trend Analysis
Month-on-Month (MoM) % change in sessions, users, and new users
Peak performance identified in February

Interactive Filtering
Dynamic slicers for Month and Channel
Enables flexible data exploration

🧠 Key Insights
Direct and Social channels are the primary drivers of traffic and user acquisition
February recorded significant growth (+115% sessions), followed by a decline in March
Low-performing channels indicate opportunities for optimisation

🛠 Tools & Technologies
Power BI Desktop
DAX (Data Analysis Expressions)
Power Query (ETL & data transformation)
Data Modelling
